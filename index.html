比起单纯的调用 API，我更热衷于拆解黑盒。无论是为了搞懂操作系统发生了什么而手动实现过 OS 内核。 在盯着 GC 日志分析内存碎片的同时，我总想知道引擎盖底下到底发生了什么。

最近正沉迷于 Agentic Workflow，试图用确定性的 Java 代码去约束不可控的概率模型。 这有点像是在试图玩一种极限运动，虽然充满挑战，但当逻辑闭环的那一刻，感觉棒极了。

我认为那些内核精密复杂、稳定的系统，其优秀的设计是互通的，是可以借鉴的。 计算机里没有黑魔法，只有严谨的工程化”......以及大量的咖啡。
